# Drapery Calculator - Documentation

## Overview
A mobile app for calculating fabric requirements for drapery curtains (Ripplefold and Pinch Pleated types).

## Features
- Calculate fabric yards, cuts, snaps based on dimensions and parameters
- Support for fractions (1/8, 1/4, etc.)
- History of calculations
- Settings with dark/light theme
- Export results (future)
- Offline support (future)

## Tech Stack
- Vue 3 + Ionic Vue 8
- Capacitor for Android/iOS
- TailwindCSS for styling
- Vuelidate for validations (future)

## Migration Notes
Migrated from Ionic Angular to Ionic Vue for modern stack and better performance.

## Deployment
- Android: `npx cap add android` then open in Android Studio
- iOS: `npx cap add ios` then open in Xcode
