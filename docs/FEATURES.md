# Features

## Core Features
- **Fabric Calculator**: Input dimensions, product type, fullness to calculate required fabric
- **Fraction Support**: Precise calculations with 1/8 inch fractions
- **Multiple Product Types**: Ripplefold (with snaps) and Pinch Pleated curtains
- **Results Modal**: Detailed breakdown of fabric needs

## Additional Features
- **Calculation History**: Save and view past calculations
- **Settings**: Dark/light theme toggle
- **Responsive UI**: Modern design with TailwindCSS

## Future Features
- Export to PDF
- Favorites configurations
- API integration for fabric prices
- Offline caching
